# example-repo

This repository contains a Python script called `album_management.py`.

## album_management.py

This script defines an `Album` class to manage album information.  
It demonstrates:

- Creating Album objects  
- Sorting albums by the number of songs  
- Swapping albums in a list  
- Copying albums between lists  
- Sorting albums alphabetically by name  
- Searching for a specific album in a list
