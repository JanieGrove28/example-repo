# example-repo

This repository contains simple Python scripts demonstrating basic programming and Git concepts.

## album_management.py

A Python script that defines an `Album` class to manage album information.  
The script shows how to:

- Create and print album objects  
- Sort albums by the number of songs or alphabetically by album name  
- Swap elements in a list  
- Search for a specific album in a list  

## hello_world.py

A simple Python script that prints messages to the console.

---

This project is designed to practice Python programming and version control with Git and GitHub.
